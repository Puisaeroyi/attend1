Building the Windows Executable
================================

This document explains how to build AttendanceProcessor.exe from source.

PREREQUISITES
-------------

• Windows 10/11 (64-bit)
• Python 3.9+ installed
• Command Prompt/PowerShell access

BUILD STEPS
-----------

1. Install PyInstaller:
   pip install pyinstaller

2. Navigate to project directory:
   cd path/to/windows_project

3. Run the build script:
   python build_exe.py

4. The executable will be created in:
   dist/AttendanceProcessor.exe

ALTERNATIVE BUILD METHOD
------------------------

If build_exe.py doesn't work, use PyInstaller directly:

pyinstaller main_gui.py ^
  --name=AttendanceProcessor ^
  --onefile ^
  --windowed ^
  --icon=icon.ico ^
  --add-data=rule.yaml;. ^
  --hidden-import=pandas ^
  --hidden-import=openpyxl ^
  --hidden-import=yaml ^
  --hidden-import=xlsxwriter ^
  --hidden-import=tkinter

VERIFYING THE BUILD
------------------

1. Check file size (should be 15-25MB)
2. Test executable on clean Windows VM
3. Verify all features work correctly

TROUBLESHOOTING
---------------

"Module not found" errors:
• Ensure all dependencies are installed
• Check hidden-imports in build script

Large file size (>30MB):
• Exclude unused modules with --exclude-module
• Check for unnecessary dependencies

Permission errors:
• Run Command Prompt as Administrator
• Check folder permissions
