Attendance & CSV Converter Tool v3.0.0 (GUI Edition)
==========================================================

A user-friendly Windows application for:
• CSV to XLSX conversion with column extraction
• Attendance data processing from biometric logs

QUICK START
-----------

1. Double-click AttendanceProcessor.exe to launch
2. Select CSV Converter tab OR Attendance Processor tab
3. Browse for input file and choose output location
4. Click Convert or Process

FEATURES
--------

✅ No Python installation required - completely standalone
✅ Point-and-click interface - no command-line knowledge needed
✅ Real-time progress feedback during processing
✅ Professional tabbed interface with menus
✅ Comprehensive error handling with helpful messages
✅ Thread-safe processing (application never freezes)

SYSTEM REQUIREMENTS
-------------------

• Windows 10 or Windows 11 (64-bit)
• 50MB free disk space
• No additional software required

CSV CONVERTER
-------------

Convert CSV files to Excel format while extracting specific columns.

1. Select "CSV Converter" tab
2. Click Browse for input CSV file
3. Click Browse for output Excel file
4. Click Convert

ATTENDANCE PROCESSOR
---------------------

Process biometric attendance data into structured attendance records.

1. Select "Attendance Processor" tab
2. Browse for input Excel file (.xlsx)
3. Choose output location for processed file
4. Keep default config (rule.yaml) or select custom config
5. Click Process

TROUBLESHOOTING
---------------

"File not found" - Check file paths and ensure files aren't open in Excel
"Permission denied" - Close output files and choose different locations
"Invalid format" - Ensure correct file types (.csv for converter, .xlsx for processor)

For detailed help, see docs/user-guide-gui.md
